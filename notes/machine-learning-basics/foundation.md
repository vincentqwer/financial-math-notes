# AI 시대의 데이터 분석과 머신러닝 개요

이 노트는 `DM_Lecure1.pdf`의 첫 강의 내용을 바탕으로, AI 시대의 데이터 분석에서 왜 머신러닝을 배워야 하는지 정리한 글이다.

핵심은 단순하다. AI는 답을 빠르게 만들어낼 수 있지만, **무엇을 물어볼지 정하고, 데이터와 모델의 결과를 판단하는 일**은 여전히 사람이 해야 한다.

## 한 줄 직관

머신러닝은 데이터를 보고 패턴을 학습한 뒤, 새로운 대상에 대해 예측하거나 데이터 구조를 설명하는 방법이다.

## AI 시대의 데이터 분석

ChatGPT 이후 AI는 데이터 분석 과정의 많은 부분을 빠르게 도와줄 수 있게 되었다. 예를 들어 모델링, 예측 정확도 개선, 분석 결과 해석, 다음 분석 방향 제안 같은 작업을 보조할 수 있다.

하지만 AI가 강해질수록 사람의 역할이 사라지는 것은 아니다. 오히려 사람은 다음 능력을 더 명확하게 가져야 한다.

- 비즈니스 문제나 연구 문제를 발견하는 능력
- 데이터를 정제하고 전처리하는 능력
- AI가 사용하는 모델의 원리를 이해하는 능력
- AI 출력 결과를 검증하는 능력
- 분석 결과를 구조화해서 전달하는 커뮤니케이션 능력
- 예측 결과의 불확실성과 의사결정 결과를 평가하는 능력

따라서 AI는 권위자가 아니라 **분석가의 능력을 증폭시키는 파트너**에 가깝다. 답을 얻는 비용은 낮아졌지만, 좋은 질문을 던지는 능력은 더 중요해졌다.

## 데이터마이닝이란 무엇인가

데이터마이닝은 많은 양의 데이터에서 유용한 정보와 패턴을 추출하는 과정이다.

데이터마이닝의 관심은 주로 **지식 발견**에 있다. 즉, 데이터 안에 숨어 있는 구조, 규칙, 관계, 이상 패턴을 찾아내는 것이 중요하다.

일반적인 데이터마이닝 과정은 다음과 같다.

1. 탐색과 전처리: 데이터를 정리하고 분석 가능한 형태로 바꾼다.
2. 패턴 식별: 데이터마이닝 기법을 사용해 패턴을 찾는다.
3. 활용: 발견한 패턴을 실제 의사결정이나 예측에 사용한다.

대표적인 기법은 다음과 같다.

- 군집화: 비슷한 패턴을 가진 그룹을 찾는다.
- 시각화: 데이터의 구조를 눈으로 확인한다.
- 연관 규칙: 변수나 항목 사이의 관련성을 찾는다.
- 이상치 탐지: 일반적이지 않은 관측값이나 패턴을 찾는다.
- 예측: 회귀나 분류를 통해 새로운 대상의 값을 예측한다.

## 머신러닝이란 무엇인가

머신러닝은 컴퓨터가 명시적인 규칙을 사람이 일일이 작성하지 않아도, 데이터의 패턴과 추론을 바탕으로 특정 작업을 수행하도록 하는 알고리즘과 통계적 모델의 집합이다.

데이터마이닝이 지식 발견에 초점을 둔다면, 머신러닝은 특히 **예측**에 초점을 둔다.

대표적인 머신러닝 기법은 다음과 같다.

- 회귀와 분류
- 군집화
- 차원 축소
- 변수 선택
- 이상치 탐지

## 데이터마이닝, 머신러닝, 통계의 관계

데이터마이닝과 머신러닝은 방법론적으로 많이 겹친다. 같은 회귀분석, 분류 모델, 군집화 기법을 서로 다른 맥락에서 사용할 수 있기 때문이다.

차이는 주된 목표에 있다.

| 분야 | 주된 목표 |
|---|---|
| 데이터마이닝 | 데이터 안에서 지식과 패턴 발견 |
| 머신러닝 | 새로운 대상에 대한 예측 |
| 통계 | 데이터 분석을 위한 모델과 추론 체계 |

머신러닝의 많은 방법은 통계학에서 출발했거나 통계적 원리를 바탕으로 한다. 그래서 통계학과 머신러닝을 함께 다루는 영역을 **통계적 학습**이라고 부른다.

## AI, 머신러닝, 딥러닝

AI, 머신러닝, 딥러닝은 서로 같은 말이 아니다.

AI는 기계가 인간의 행동을 모방하거나 지능적인 작업을 수행하도록 만드는 넓은 분야다. 규칙 기반 시스템도 AI의 한 형태로 볼 수 있다.

머신러닝은 AI 안에 포함되는 방법이다. 데이터에서 패턴을 학습하고, 그 패턴을 이용해 예측이나 의사결정을 수행한다.

딥러닝은 머신러닝 안에 포함되는 방법이다. 여러 층으로 쌓인 신경망을 계산 가능하게 만들고, 이미지, 텍스트, 음성 같은 복잡한 비정형 데이터에서 강한 성능을 보인다.

```txt
AI
└── Machine Learning
    └── Deep Learning
```

## 빅데이터와 머신러닝

빅데이터는 머신러닝과 딥러닝 모델의 예측 정확도를 크게 끌어올렸다.

빅데이터는 전통적인 표 형태의 데이터뿐 아니라 이미지, 텍스트, 음성 같은 비정형 데이터까지 포함한다. 데이터가 많아지면 계산 문제가 커지고, 특히 변수의 수가 매우 많은 고차원 데이터에서는 통계적으로도 어려운 문제가 생긴다.

그래서 현대 데이터 분석에서는 단순히 데이터가 많은 것보다, 어떤 변수를 어떻게 만들고, 어떤 구조를 모델이 학습하게 할 것인지가 중요하다.

## 통계적 학습의 목표

이 강의의 관점에서 핵심은 **통계적 학습**이다.

통계적 학습은 데이터에서 배우는 방법이다. 목적은 크게 두 가지로 볼 수 있다.

1. 예측: 새로운 대상의 결과를 맞힌다.
2. 구조 설명: 데이터 안의 구조나 관계를 이해한다.

이 목적에 따라 학습 문제는 크게 세 가지로 나뉜다.

## 지도학습

지도학습은 결과 변수와 설명 변수가 모두 있는 상황이다.

예를 들어 기존 고객의 금융 이력과 신용점수가 있다면, 금융 이력은 입력이고 신용점수는 출력이다. 이 데이터를 바탕으로 모델을 만들고, 새 고객의 금융 이력을 넣어 신용점수를 예측할 수 있다.

지도학습의 기본 절차는 다음과 같다.

1. 학습 데이터 준비: 결과값과 입력 변수가 함께 있는 데이터를 준비한다.
2. 예측 모델 구축: 학습 데이터를 사용해 모델을 만든다.
3. 새 대상 예측: 아직 결과를 모르는 새로운 대상에 대해 예측한다.

용어는 분야마다 조금 다르게 쓰인다.

| 통계 | 머신러닝 | 의미 |
|---|---|---|
| 종속변수 | output, outcome | 예측하고 싶은 결과 |
| 독립변수 | input, feature | 예측에 사용하는 정보 |

대표적인 지도학습 방법에는 회귀분석, 판별분석, 로지스틱 회귀, 의사결정나무, 신경망 등이 있다.

## 비지도학습

비지도학습은 결과 변수가 없고 입력 변수만 있는 상황이다.

목표는 정답을 맞히는 것이 아니라 데이터의 구조를 설명하는 것이다. 예를 들어 고객 데이터를 보고 비슷한 고객군을 나누는 군집분석이나, 많은 변수를 적은 수의 축으로 요약하는 주성분분석이 여기에 들어간다.

## 강화학습

강화학습은 환경과 상호작용하면서 보상을 통해 행동 방식을 학습하는 방법이다.

완전한 정답이 매번 주어지는 것이 아니라, 어떤 행동이 좋은 결과를 가져왔는지에 대한 보상 정보를 바탕으로 점점 더 나은 전략을 찾는다. 알파고 같은 사례가 대표적이다.

## 응용 분야

데이터마이닝과 머신러닝은 다양한 분야에서 사용된다.

- 마케팅: 고객 세분화, 타깃 고객 선정, 맞춤형 프로모션
- 신용평가: 고객의 신용 상태나 신용점수 예측
- 바이오인포매틱스: 유전체 데이터에서 정보 추출
- 이상 탐지: 금융 사기 탐지, 공장 장비 고장 예방

## 요약

- 데이터마이닝은 데이터에서 유용한 패턴과 지식을 찾는 과정이다.
- 머신러닝은 데이터에서 패턴을 학습해 새로운 대상에 대해 예측하는 데 초점을 둔다.
- 통계학은 머신러닝과 데이터마이닝의 중요한 방법론적 기반이다.
- AI, 머신러닝, 딥러닝은 포함 관계에 있다.
- 지도학습은 outcome과 feature가 모두 있고, 비지도학습은 feature만 있다.
- AI 시대에는 모델을 쓰는 능력뿐 아니라 문제 설정, 전처리, 검증, 커뮤니케이션 능력이 중요하다.
